Pizza Sales Analysis – Power BI, Python & MongoDB Project
1. Project Overview

This project focuses on analyzing pizza sales data to understand overall business performance, customer behavior, and sales trends. It follows a complete data pipeline starting from raw data to dashboard visualization.

The project also includes a GUI-based system for managing orders, making it closer to a real-world application.

2. Objective

Analyze sales performance and revenue trends

Identify top-selling categories and sizes

Understand customer ordering patterns

Track peak order hours

Build a complete data pipeline using modern tools

3. Project Workflow
Step 0 – Data Collection

The dataset was taken from Maven Analytics (Pizza Sales dataset), which includes order details, pizza categories, sizes, quantity, and timestamps.

Step 1 – Database Setup (MongoDB Atlas)

Created a database in MongoDB

Uploaded the dataset

Used MongoDB Atlas for cloud-based access

Step 2 – GUI Development (Tkinter)

Built a GUI using Python Tkinter

Features:

Insert new orders

Manage data easily

Simulates a real-time order entry system

Step 3 – Dashboard Creation (Power BI)

Built an interactive dashboard in Power BI

Used filters like category and date range

Designed a visually appealing layout for insights

4. Tools Used

Python (Tkinter,Pandas)

MongoDB Atlas

Power BI

5. Dashboard Preview

The dashboard provides a complete overview of pizza sales performance:

KPIs: Revenue, Total Orders, Quantity Sold, Avg Order Value

Sales by Category (Classic, Supreme, Veggie, Chicken)

Sales by Size (L, M, S, XL, XXL)

Peak Hour Analysis

Worst Selling Products

Revenue Distribution

6. Key Insights

Large size pizzas contribute the highest sales

Classic category generates the most revenue

Peak order time is around afternoon to evening

Some products have very low sales and can be optimized or removed

7. Conclusion

This project demonstrates a complete data analytics workflow including data storage, application layer, and visualization. It shows how raw data can be transformed into meaningful business insights using multiple tools.
